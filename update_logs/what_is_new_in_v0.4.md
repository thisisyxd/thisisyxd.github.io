# v0.4 — 导航地铁视觉增强

## 新增功能

### 1. 导航时地铁样式动态变化
- 导航激活时，地铁线路按乘坐状态自动调整样式：
  - **未乘坐线路** → 整条线路变为浅灰色（`#666`，`opacity: 0.6`）
  - **已乘坐线路** → 逐段绘制覆盖层，已乘段加粗（`stroke-width: 1.5`），未乘段灰色
  - **未访问车站** → 所在城市有地铁运行时，半透明变暗（`opacity: 0.5`）
  - **关键站**（段端点 + 使用的换乘站） → 放大至 `r=3`
  - **普通经过站** → 放大至 `r=2`
- 换乘站样式规则：
  - 实际发生换乘 → 保持白色填充，`stroke-width: 1.5`
  - 上车/下车（非换乘）或仅经过 → 切换为普通站样式（线路色填充、细描边）
- 退出导航（清除/切换模式）时所有样式自动恢复

### 2. 起点/终点标记缩放优化
- 缩放至城市名称隐藏时（`zoom > 1.8`），标记半径从 `r=8` 缩小至 `r=3`（约原尺寸 1/3）
- 使用行内样式（`style.r`）替代属性（`setAttribute`），确保 CSS 类不覆盖缩放
- 脉冲动画同步缩小（`pulse-sm`：`6→12`）

### 3. 普通站尺寸调整
- 所有非换乘站默认半径从 `r=1` 改为 `r=1.5`

## 移除功能

### 1. 换乘次数统计
- 删除 `metroTransferCount` 计数逻辑
- 路线信息文本不再显示"需换乘N次"，仅保留"（需乘地铁）"

## 文件变更

| 文件 | 变更 |
|------|------|
| `app.js` | 新增地铁导航样式系统（`resetMetroNavStyles`/`applyMetroNavStyles`）；路线绘制时调用样式；`initMetro` 中为车站/线路添加 `data-city`、`data-station`、`data-line` 属性；标记缩放改用行内样式；删除换乘统计；普通站默认半径改为 1.5 |
| `style.css` | `pulse-sm` 动画从 `8→16` 调整为 `6→12` |

## 核心算法/思想

**导航时地铁样式分层系统**：
- 未乘坐线路：保留原路径，直接修改 `stroke` 和 `opacity` 属性
- 已乘坐线路：隐藏原路径，按序列逐段绘制覆盖层（`metro-nav-overlay`）
  - 每段独立判断是否被路径经过（通过比对 `nodePath` 中的边）
  - 已乘段：原始颜色 + 粗描边；未乘段：灰色 + 常规描边
- 车站样式：通过 `style.*` 行内样式覆盖 CSS 类，确保优先级正确
  - 使用 `keyStations` 和 `transferStations` 两集合分别控制放大和换乘UI
  - 段端点（含上下车站）和换乘站 → `r=3`
  - 普通经过站 → `r=2`
  - 未访问站 → 半透明
