# Minecraft 世界导航地图

基于 Minecraft 1.7.2 创建的世界，种子 `6220807999717207113`，游玩时间约 **280 小时**，在游玩至约 240 小时时升级至 **1.21** 版本。

## 服务器信息

- **核心**：Fabric
- **模式**：生存模式 + 创造模式（主要），仅允许手动放置方块
- **Mods**：`fabric-api`、`dynmap`、`chunky`（均为提升体验的辅助模组）
- **风格**：主打原版体验

## 世界布局

世界分为**生存区**与**创造区**，二者相互独立、互不影响。整体大小约 **3500×4000**，边界限制活动范围约 **5000×5000**。

> ⚠️ 请不要在生存时进入创造区，也不要在创造时进入生存区！

### 出生点

坐标 `(0, 0)`，周围几十格范围的方块设置为**不可破坏**。正下方为一个红石时钟，用于计时服务器运行的真实时间。出生后可选择前往生存区（**不可逆**）或创造区。

### 生存区域

- 主要位于 `x < -700`
- 可能传送至：`(-900, -570)`（靠近创造区，开发程度适中）或 `(-1500, -900)`（开发程度较低）
- 地形：黑森林、平原、森林、沼泽、海洋

### 创造区域

- 主要位于 `x > -700`
- 地形：沙漠、热带草原、河流及少量平原和湖泊
- 此地图**仅包含创造区**的情况

## 项目结构

```
├── index.html               # 入口页面
├── style.css                 # 样式
├── roads-builder.js          # 道路/铁路/地铁数据构建逻辑
├── app.js                    # 地图渲染、导航、寻路等应用逻辑
├── README.md
└── roads/
    ├── citys/
    │   └── city_location.js  # 城市中心坐标
    ├── roads/
    │   ├── A.js ~ H.js       # 各城市道路坐标
    │   └── connection_road.js# 额外连接道路
    ├── railway/
    │   └── railway.js        # 铁路车站、中间点、线路
    └── metro/
        └── metro.js          # 地铁系统数据（预留）
```

- **数据文件**（`roads/` 下各文件）：仅包含坐标等纯数据，修改后刷新即可生效
- **逻辑文件**（`roads-builder.js`、`app.js`）：负责将数据构建为可用的图结构并渲染
